Moon simulation

月の満ち欠け

85th 正位置の愚者
新月から満月までのシュミレーションを作りました
新機能として見たい月の時止められる機能と、朝と昼と夜の三場面を見ることができます。
コードもgithubに公開しています。


使用したツール

  • Processing

補足(by 0x4C)

今回の月のシミュレーションは、ProcessingのP3Dモードという3Dの図形を描画するための機能を使って実装しました。
より具体的に説明すると、公転運動のような動きをする球にカメラを向けることによって月の満ち欠けをシミュレートしています。
